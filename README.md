# 证件照 4×6 自动排版工具

一个运行在 macOS 桌面的证件照裁切与打印排版工具。导入照片后，可按常用证件照尺寸裁切、微调人像，并生成适合 4×6 英寸相纸打印的 300 DPI JPG。

图片只在本机处理，不会上传到网络。

## 主要功能

- 生成 4×6 英寸（1200×1800 px，300 DPI）排版图
- 支持一寸、小一寸、大一寸、二寸、小二寸、驾照、结婚照及自定义尺寸
- 自动识别人像主体，并预留头顶空间
- 可调整缩放、左右位置、上下位置、数量、行列和间距
- 自动或手动选择排版行列
- 可添加并延长裁切辅助线
- 实时预览单张照片和整版效果
- 一键保存单张证件照与排版图
- 调用 macOS“预览”直接进入打印流程
- 可自选输出文件夹，并记住上次设置

## 运行环境

- macOS
- Python 3.10 或更高版本
- Tkinter（macOS 官方 Python 安装包通常自带）
- Pillow 10 或更高版本

## 安装与启动

1. 下载或克隆本仓库：

   ```bash
   git clone https://github.com/pandastone9-eng/id-photo-4x6-layout-tool.git
   cd id-photo-4x6-layout-tool
   ```

2. 安装依赖：

   ```bash
   python3 -m pip install -r requirements.txt
   ```

3. 双击 `一键生成证件照.command`，或在终端运行：

   ```bash
   python3 id_photo_tool.py
   ```

## 使用方法

1. 点击“选择照片”。
2. 选择证件照尺寸和排版张数。
3. 按需调整行列、间距、裁切位置与缩放。
4. 在右侧确认单张和 4×6 排版预览。
5. 点击“保存单张和排版”，成品默认保存到桌面的 `证件照输出` 文件夹。

也可以点击“选择输出文件夹”修改保存位置。软件会在用户目录保存一个 `.id_photo_print_tool_config.json` 配置文件，用来记住上一次的输出路径。

## 支持的照片尺寸

| 类型 | 尺寸 |
| --- | --- |
| 一寸 | 25×35 mm |
| 小一寸 / 驾照 | 22×32 mm |
| 大一寸 | 33×48 mm |
| 二寸 | 35×49 mm |
| 小二寸 / 小两寸 | 35×45 mm |
| 结婚照 | 53×35 mm |
| 自定义 | 可输入宽度和高度 |

## 打印设置

- 纸张选择：4×6 英寸（约 10.2×15.2 cm）
- 缩放选择：100% / 实际大小
- 不要选择“适合页面”，否则证件照实际尺寸会发生变化

## 命令行批量处理

也可以把一个或多个图片路径作为参数传入：

```bash
python3 id_photo_tool.py photo1.jpg photo2.png
```

命令行模式默认生成二寸（35×49 mm）、6 张排版，并保存到当前设置的输出文件夹。

## 隐私说明

本工具没有后台服务，也不包含网络上传逻辑。照片的读取、裁切、排版和导出均在本机完成。
